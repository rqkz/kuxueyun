## 有哪些坑需要注意？

没有完美的商家，酷雪云也有一些需要注意的地方。

第一是IP。部分机房的IP可能进过黑名单，尤其是那些低价套餐的共享IP。如果你对IP质量要求高（比如做邮件服务器、跑广告），最好提前测试或者买独立IP。

第二是CPU限制。有些套餐标注的是"Fair Use"（合理使用），意思是CPU不能长期满载。如果你要跑计算密集型任务，最好选那些明确标注独享CPU的套餐，或者提前问客服。

第三是退款政策。👉 [特价促销套餐都是不退款的](https://kuxueyun.com/aff.php?aff=101)，这个要特别注意。正价套餐有些支持退款，但要扣手续费，具体得看服务条款。

第四是国内线路的实名要求。如果你买北京机房的套餐，需要提供身份证、手机号等信息做实名认证。介意隐私的话，可以选海外机房。
